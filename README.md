#  Учебный помощник (Python + Алиса)

##  Описание проекта

Учебный помощник — это веб-приложение и навык для голосового ассистента Алиса, предназначенный для изучения языка Python.
Пользователь может получать объяснения тем, проходить тестирование и отслеживать свой прогресс.

Проект объединяет:

* веб-интерфейс (админ-панель)
* базу данных
* REST API
* навык Алисы (через webhook)

---

##  Основные возможности

###  Для пользователя (Алиса)

* объяснение тем Python (переменные, циклы, функции и т.д.)
* прохождение тестирования
* проверка ответов
* подсчёт результата
* отслеживание прогресса
* работа с контекстом диалога

---

###  Для администратора (веб-интерфейс)

* авторизация
* просмотр списка вопросов
* удаление вопросов
* загрузка вопросов из JSON-файла
* управление базой данных

---

##  Архитектура проекта

```text
Клиент (браузер / Алиса)
        ↓
Flask-приложение
        ↓
┌───────────────┐
│ routes        │ → обработка запросов
│ services      │ → бизнес-логика
│ models        │ → работа с БД
└───────────────┘
        ↓
SQLite база данных
```

---

##  Используемые технологии

* Python
* Flask
* Flask-SQLAlchemy (ORM)
* Flask-Login (авторизация)
* Bootstrap + CSS (интерфейс)
* REST API
* Webhook (интеграция с Алисой)
* Requests (внешний API — цитаты)
* SQLite (хранение данных)
* Render (хостинг)

---

##  API

Примеры доступных endpoints:

* GET /api/questions — получить список вопросов
* GET /api/progress/<user_id> — получить прогресс пользователя

---

##  Навык Алисы

Навык взаимодействует с сервером через webhook:

POST /webhook

Поддерживаемые команды:

* "привет"
* "хочу тест"
* "объясни список"
* "мой прогресс"

##  Запуск проекта (локально)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/alice-study-assistant.git
cd alice-study-assistant
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
source .venv/bin/activate   # Linux/Mac
.venv\Scripts\activate      # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python app.py
```

Открыть в браузере:

```text
http://127.0.0.1:5000
```
