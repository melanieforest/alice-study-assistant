# Алиса: Учебный помощник

## Описание проекта

«Алиса: Учебный помощник» — это навык для голосового ассистента Алиса, предназначенный для интерактивного обучения пользователей по языку программирования Python.

Проект позволяет объяснять темы, проводить тестирование, проверять ответы пользователя, сохранять прогресс и вести диалог с учётом контекста. Серверная часть реализована на Python с использованием Flask.

## Основные возможности

- объяснение тем по Python;
- проведение тестов;
- смешанный формат вопросов: открытые и закрытые;
- проверка ответов;
- хранение прогресса пользователя;
- работа с контекстом диалога;
- административная панель;
- авторизация администратора;
- загрузка учебных материалов из JSON;
- интеграция со сторонним API.

## Архитектура проекта

Проект состоит из следующих основных модулей:

- `app.py` — точка входа в приложение;
- `routes/` — маршруты приложения;
- `models/` — ORM-модели базы данных;
- `services/` — бизнес-логика проекта;
- `templates/` — HTML-шаблоны админ-панели;
- `static/` — стили и скрипты;
- `data/` — учебные материалы и вопросы;
- `uploads/` — загружаемые файлы.

## Основные сценарии работы

### 1. Запуск навыка
Пользователь запускает навык, после чего система предлагает выбрать режим работы:
- обучение;
- тестирование;
- просмотр прогресса.

### 2. Обучение
Пользователь выбирает тему, и навык объясняет материал простым языком.

### 3. Тестирование
Навык задаёт вопросы по Python, принимает ответ пользователя, проверяет его и сохраняет результат.

### 4. Работа с контекстом
Система сохраняет текущее состояние диалога:
- выбранную тему;
- режим работы;
- текущий вопрос;
- ожидание ответа.

### 5. Административная панель
Администратор может:
- войти в систему;
- просматривать список вопросов;
- загружать новые вопросы;
- управлять учебными материалами.

## Структура базы данных

В проекте предполагаются следующие сущности:

- `User` — пользователь;
- `Topic` — тема;
- `Question` — вопрос;
- `AnswerAttempt` — попытка ответа;
- `SessionState` — текущее состояние диалога.

## Работа с API

В проекте предусмотрено использование внешнего API для генерации объяснений, подсказок и комментариев к ответам пользователя.
